# Generateive_AI_Research
## Project Overview 
This repository contains all data, documents, and scripts for the research project "Using Generative AI Tools - Boon or Bane" The project investigates the use of generative AI tools and includes both quantitative (survey) and qualitative (interview) data, along with literature, analysis scripts and reports.

## File Naming Convention 
All files in this repository follow this naming convention:
```
YYYYMMDD_ProjectAbbreviation_Description_Version.extension
```

**Example:** `20260429_GenAI_SurveyData_v1.csv`

- `YYYYMMDD` — Date the file was created or last updated
- `GenAI` — Project abbreviation
- `Description` — Short description of the file contents
- `v1` — Version number (update to v2, v3 etc. when making major changes)

---

## Version Control Guidelines

- Always create a **new branch** for your changes — do not edit the main branch directly
- Write **meaningful commit messages** that clearly describe what was changed and why
- Update the **version number** in the file name when making significant changes (e.g., `v1` → `v2`)
- Resolve any **merge conflicts** carefully before submitting a pull request

---

## How to Contribute

1. **Fork** this repository to your own GitHub account
2. **Clone** your forked repository to your local machine using GitHub Desktop or the terminal
3. Create a **new branch** with a descriptive name (e.g., `add-survey-analysis-script`)
4. Make your changes and **commit** them with a clear, descriptive commit message (e.g., `Added cleaned survey data and initial analysis scripts`)
5. **Push** your branch and submit a **Pull Request** to the main repository
6. The repository owner will **review** your pull request and merge it if approved
